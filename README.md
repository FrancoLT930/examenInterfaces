##Edo Sushi Reservas

Edo Sushi Reservas es una aplicación web desarrollada con React que permite a los clientes realizar reservas de mesas en el restaurante Edo.  
El usuario puede seleccionar la sede, la cantidad de personas, la fecha y la hora de su visita, además de indicar si celebra una fecha especial o presenta alguna restricción alimentaria.

---

##Funcionalidades principales

- Selección de sede del restaurante.  
- Elección de cantidad de personas.  
- Selección de fecha y hora de reserva.  
- Opción para indicar fecha importante o restricción alimentaria.  
- Interfaz visual moderna y sencilla de usar.  

---

##Tecnologías utilizadas

- **React** – Framework principal para el desarrollo de la interfaz.  
- **HTML5** y **CSS3** – Estructura y estilos.  
- **JavaScript (ES6)** – Lógica de la aplicación.  
- **Local Storage** – Almacenamiento temporal de los datos de reserva.
